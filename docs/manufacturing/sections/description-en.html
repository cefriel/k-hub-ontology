The Knowledge Hub Ontology - Manufacturing: Description back to ToC

This section describes the classes and properties in detail through a textual description and a diagram of the module and the named individuals included in the thesaurus.

The Manufacturing module consists of 12 concepts modelled as subclasses of the most general concept [kh-a:Topic](https://knowledge.c-innovationhub.com/k-hub/annotation#Topic) defined in the [Annotation module](https://knowledge.c-innovationhub.com/k-hub/annotation). These subclasses describe general maintenance elements such as: _Component_, _Configuration_, _Supplier_, _DocumentType_, _Trouble_, _Action_, _Product_, _Machine_, _Workstation_, _SparePart_, _Error_ and _Tool_.

This module includes also a taxonomy of instances of the aforementioned concepts. This taxonomy represents a complete list of entities to be searched in the annotation of documents which are specific to the domain of interests. The thesaurus contains a list of terms translated into English and Italian and a list of possible synonyms. The thesaurus was modeled using SKOS, a W3C recommendation designed for the representation of thesaurus, classification schemes and taxonomies.
